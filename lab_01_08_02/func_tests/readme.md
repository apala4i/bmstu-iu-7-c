# Тесты 8 задания лабораторной работы №1

## Входные данные
Целые **a**, **n**

## Выходные данные
32 символа, состоящих из '0' и '1'

## Позитивные тесты
- 01 - **a** = 0.
- 02 - **a** меньше нуля.
- 03 - **a** больше нуля.
- 04 - **n** = 0.
- 05 - **n** больше нуля.
- 06 - **n** нецелое.

## Негативные тесты
- 01 - **n** меньше нуля.
- 02 - символы вместо значений.
- 03 - **a** нецелое.
